# RECOVER Gangnam Project Memory

Last updated: 2026-05-01

## Project Role

This project is not a simple homepage production project.

It must operate as the **RECOVER Gangnam Brand Department**, responsible for building a world-class brand strategy, message system, customer experience, and market-creating category around Korean skin recovery.

## Core Brand Definition

리커버 강남은 피부가 돌아오는 조건을 설계하는 한의 피부재생 브랜드입니다.

## Market Creation Thesis

리커버 강남은 강남 미용 시장에서 "더 빠른 시술", "더 싼 이벤트", "더 많은 전후사진"으로 경쟁하지 않는다.

리커버 강남이 만들어야 할 시장은:

> 강남에서 피부를 더 하기 전에, 회복의 조건을 먼저 점검하는 시장.

## Core Brand Language

- Skin Recovery Design
- 피부 회복 설계
- 시술보다 먼저 회복의 조건
- 회복은 속도가 아니라 조건
- 강남에서 피부를 더 하기 전에, 회복의 조건부터

## Operating Method

- Read: 피부의 결, 흔적, 균형, 생활 리듬을 먼저 읽는다.
- Restore: 더 많이 자극하기보다 피부가 회복될 환경을 정돈한다.
- Record: 변화를 기록하고 다음 회복의 순서를 조정한다.

## Strategic Rule

Every homepage, content, channel, and consultation decision should strengthen this equation:

> RECOVER Gangnam = Skin Recovery Design = 피부가 돌아오는 조건을 설계하는 한의 피부재생 브랜드

Avoid drifting back into generic clinic language such as price events, luxury claims, equipment-first messaging, or before/after-only persuasion.
